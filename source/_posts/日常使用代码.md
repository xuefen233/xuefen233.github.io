npm install hexo -g 安装hexo命令
hexo new(n) "文章名称" 新建一篇标题为XXX的文章
hexo server(s) 启用本地服务器，用来预览主题。默认地址：http://localhost:4000/
hexo clean 清除缓存文件db.json和已生成的静态文件public
hexo deploy(d) 自动生成静态文件，并部署到设定仓库
hexo generate(g) 生成静态文件到默认的public文件夹

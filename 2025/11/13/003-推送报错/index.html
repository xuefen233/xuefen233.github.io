推送到Github报错

报错信息

检查ssh公钥有没有连上GitHub

需要开代理或加速服务，开了之后再去重新生成 ssh ，填到你的GitHub上面去，最后网站就能正常打开了

生成ssh公钥

yourname=GitHub账户名，youremail=注册邮箱。

生成公钥，一路回车

然后就可以在C/用户/“用户名”/目录下生成一个 .ssh 的文件，打开它，里面会有一个 id_rsa.pub 的文件。

打开 GitHub ，点击 settings 设置界面，找到 SSH and GPG keys 的设置选项，点击 New SSH key ，把 id_rsa.pub 里面的内容复制进去，查看是否成功。
